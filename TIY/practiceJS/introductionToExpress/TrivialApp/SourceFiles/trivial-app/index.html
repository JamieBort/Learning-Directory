App Configuration

Default Route

Listen on root

Serve index.html

app.get('/', function (req, res) { res.sendFile(path.join(__dirname + '/index.html')); })

Set Static Files In A Directory

configure express.static to use a virtual prefix (/files) and an absolute path.

app.use('/files', express.static(path.join(__dirname, 'files')));

Listen On Port 3000

app.listen(3000, function () { console.log('Successfully started node application!') })
